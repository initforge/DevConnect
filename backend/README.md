# DevConnect Backend

This service provides the REST API for the DevConnect mobile application.

## Quick Start
For detailed instructions on setup, testing, and development, please refer to the main project documentation:
- **[Development Guide](../docs/DEVELOPMENT.md)**: Setup, running scripts, and testing.
- **[System Architecture](../docs/ARCHITECTURE.md)**: Technical design and API specifications.

## Technology Stack
- **Runtime**: Node.js v20+
- **API**: Custom HTTP Server
- **Database**: PostgreSQL 16
- **Real-time**: WebSockets
- **Deployment**: Docker & Docker Compose
